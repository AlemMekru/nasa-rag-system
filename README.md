# NASA RAG System

A Retrieval-Augmented Generation (RAG) system that answers questions about NASA mission transcripts using semantic search and large language models.

## Overview

This project demonstrates how Retrieval-Augmented Generation can be used to improve question-answering over large document collections.

The system retrieves relevant mission documents using vector embeddings and then generates answers using a language model.

## Features

- Semantic search using embeddings
- Vector database retrieval
- Context-aware answer generation
- Evaluation of responses

## Technologies

- Python
- Large Language Models (LLMs)
- Vector Databases
- Retrieval-Augmented Generation (RAG)

What evidence explains the Challenger failure?
